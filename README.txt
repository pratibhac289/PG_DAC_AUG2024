                              
                                          Project Name-Bank_system
Added functionalities-

1) Create user account.
2) Deposite balance
3) Withdraw balance
4) Delete user account

Project Structure

main.cpp:
 This is the main file that runs the system and allows users to interact with the bank management system.
create_account.cpp:
 This file handles the creation of new accounts.
deposit_money.cpp: 
  This file manages the deposit of money into the created accounts.
bank_system.h: 
  Header file containing the Account class definition and function declarations

Execution-

1)create the new project.
2)add this all files in this project.
3)run the project.
